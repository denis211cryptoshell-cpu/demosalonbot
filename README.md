# 💇‍♀️ Salon Bot — Telegram-бот для записи в салон красоты

Бот для автоматизации записи клиентов в салон красоты с сохранением данных в Google Sheets.

---

## 🚀 Возможности

- **Запись на услуги** — пошаговый процесс выбора услуги, ввода имени, телефона и даты визита
- **Google Sheets интеграция** — все записи автоматически сохраняются в таблицу
- **FSM (Finite State Machine)** — управление состояниями диалога
- **Inline и Reply клавиатуры** — удобный выбор услуг и навигация
- **Отмена записи** — возможность отменить запись на любом этапе

---

## 📋 Функционал

### Команды и кнопки

| Команда / Кнопка | Описание |
|------------------|----------|
| `/start` | Запуск бота, главное меню |
| `/cancel` | Отмена текущего действия |
| ✂️ Записаться | Начать процесс записи на услугу |
| 📋 Мои записи | Просмотр своих записей (в разработке) |
| ❌ Отменить запись | Отмена существующей записи |

### Процесс записи

1. **Выбор услуги** — клиент выбирает услугу из списка:
   - Стрижка
   - Окрашивание
   - Маникюр
   - Педикюр

2. **Ввод имени** — клиент вводит своё имя

3. **Ввод телефона** — клиент вводит номер телефона

4. **Выбор даты и времени** — клиент указывает желаемое время визита

5. **Подтверждение** — бот показывает детали записи и сохраняет в Google Sheets

## 🛠 Технологии

- **Python 3.11+**
- **aiogram 3.x** — асинхронный фреймворк для Telegram Bot API
- **gspread** — работа с Google Sheets API
- **google-auth** — авторизация в Google Cloud
- **python-dotenv** — загрузка переменных окружения

## ⚙️ Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/denis211cryptoshell-cpu/demosalonbot.git
cd demosalonbot
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Настройка переменных окружения

Создайте файл `.env` в корне проекта:

```env
BOT_TOKEN=ваш_токен_бота
GOOGLE_SPREADSHEET_ID=id_таблицы_google_sheets
GOOGLE_SERVICE_ACCOUNT_FILE=credentials.json
ADMIN_ID=ваш_admin_id (опционально)
```

### 4. Настройка Google Sheets

1. Создайте новую Google Таблицу
2. Скопируйте ID таблицы из URL (между `/d/` и `/edit`)
3. Создайте сервисный аккаунт в [Google Cloud Console](https://console.cloud.google.com/)
4. Скачайте JSON-ключ и сохраните как `credentials.json`
5. Откройте доступ к таблице для email сервисного аккаунта (права **Редактор**)

### 5. Запуск бота

```bash
python main.py
```

## 🌐 Развёртывание

### Bothost

1. Создайте нового бота в панели Bothost
2. Укажите Git-репозиторий: `https://github.com/denis211cryptoshell-cpu/demosalonbot.git`
3. Добавьте переменные окружения (см. выше)
4. Загрузите файл `credentials.json` через файловый менеджер
5. Запустите бота

### Docker (опционально)

```dockerfile
FROM python:3.11-slim

WORKDIR /app

COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

COPY . .

CMD ["python", "main.py"]
```
